プライバシーポリシー

NewsRadarJP — 最終更新日: 2026年3月23日

1. サービスの概要

NewsRadarJP（以下「当サービス」）は、国際ニュースを短尺動画として自動生成・配信する個人運営のメディアプロジェクトです。YouTube、X（旧Twitter）、Threads、Instagram で動画およびテキストコンテンツを配信しています。

2. 収集する情報

当サービスは、ユーザーの個人情報を直接収集・保存・処理しません。

  • アカウント登録機能はありません
  • Cookie やトラッキングツールは使用していません
  • ユーザーの閲覧履歴、位置情報、デバイス情報等を独自に収集することはありません

3. YouTube API Services の利用

当サービスは、YouTube Data API v3 を使用して動画コンテンツをアップロードしています。YouTube API Services の利用にあたり、YouTube 利用規約および Google プライバシーポリシーが適用されます。

当サービスは YouTube API を通じてユーザーデータにアクセスしたり、収集・保存したりすることはありません。API はコンテンツのアップロードのみに使用されます。

4. 第三者プラットフォーム

当サービスのコンテンツは以下のプラットフォームで配信されます。各プラットフォームにおける情報の取り扱いは、それぞれのプライバシーポリシーに従います。

  • YouTube（Google）
  • X（旧Twitter）
  • Instagram（Meta）
  • Threads（Meta）
  • TikTok（ByteDance）

5. データの保持と削除

当サービスはユーザーの個人データを保持していないため、削除対象となるデータはありません。YouTube API Services を通じて取得されたデータも保持しておりません。

Google アカウントに関連するデータの管理については、Google セキュリティ設定から第三者アプリのアクセス権を確認・取り消すことができます。

6. お問い合わせ

本プライバシーポリシーに関するお問い合わせは、以下の方法でご連絡ください。

  • X（旧Twitter）: @NewsRadarJP

7. ポリシーの変更

当サービスは、必要に応じて本プライバシーポリシーを更新することがあります。変更があった場合は、本ページに掲載することでお知らせいたします。
